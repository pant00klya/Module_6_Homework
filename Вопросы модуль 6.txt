1. Дайте определения понятиям: класс, конструктор, параметры конструктора, метод.
Класс — это модель для создания объектов определённого типа, описывающая их структуру и определяющая алгоритмы
для работы с этими объектами.

Конструктор — это специальный набор инструкций (метод), который вызывается только при создании объекта.

Параметры — это те же самые переменные, которые задаются при создании объекта (при вызове конструктора). В этих переменных лежат значения,
которые необходимо использовать для инициализации полей объекта, что мы и делаем в теле конструктора.

Метод — это набор инструкций, которые могут быть вызваны у объекта.

2. Сколько конструкторов может быть у класса?
Количество конструкторов у класса не ограничено.

3. Что такое сигнатура метода? Является ли возвращаемое значение частью сигнатуры?
Сигнатура метода - это имя метода и входящие параметры.

4. Что такое void? Для чего он нужен?
Методы void не возвращают результат выполнения инструкций в код, в котором он был вызван

5. Что такое область видимости? Какие области видимости существуют? Чем может быть ограничена область видимости?
Область видимости — это части кода, откуда переменная может быть использована.

6. Для чего нужно слово this?
This — это специальное ключевое слово, которое указывает, что данное поле — это поле объекта нашего класса.
